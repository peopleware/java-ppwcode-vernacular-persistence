Hibernate 3 implementations of DAO's. The dependency on Hibernate 3 of this library however is optional. As long as you don't use Hibernate 3 related functionality from this library, you do not need Hibernate 3 libraries in your project.

Note: This functionality is ported from an earlier version, and refactored during that port. At the time of this writing, it has not been used in this form yet in a project.
